# 使用指南 - 今天吃什么

## 🎯 快速开始

### 方法1：本地使用
直接双击 `index.html` 文件，即可在浏览器中打开应用。

### 方法2：手机使用
1. 将整个文件夹复制到手机
2. 用手机浏览器打开 `index.html`
3. 或者使用 HTTP 服务器部署后访问

---

## 📱 功能详解

### 1. 转盘抽取页面

**基本操作：**
- 点击"开始抽取"按钮，转盘会旋转3-4秒后停下
- 指针指向的美食就是今天的推荐！
- 查看结果卡片，包含美食图片、名称、分类和描述

**筛选功能：**
- 点击分类标签（如"中式快餐"、"面食"）可以只从该分类中抽取
- 可以同时选择多个分类
- "全部"复选框选中时，从所有美食中抽取

**结果操作：**
- ⭐ 收藏：将喜欢的美食加入收藏夹
- 🔄 再抽一次：不满意？继续抽！

**手机专属：**
- 🎲 摇一摇手机即可触发抽取（需要支持陀螺仪的手机）

---

### 2. 列表页面

**浏览美食：**
- 查看所有65种美食的完整列表
- 每张卡片显示美食图片、名称和分类

**搜索功能：**
- 在搜索框输入关键词（如"鸡"、"面"）
- 实时过滤显示匹配的美食

**分类筛选：**
- 点击顶部的分类按钮快速切换
- 15个分类任意选择

**收藏操作：**
- 点击卡片右上角的 🤍 图标可收藏
- 已收藏的会显示 💖

---

### 3. 收藏页面

- 查看所有你收藏的美食
- 点击取消收藏图标可移除
- 点击卡片查看详情

---

### 4. 历史记录页面

- 查看最近50次的抽取记录
- 显示美食名称、分类和抽取时间
- 点击"清空历史"按钮可删除所有记录
- 点击历史项可跳转到详情

---

## 🌙 深色模式

- 点击右上角的 🌙 按钮切换到深色模式
- 再次点击（显示 ☀️）切换回浅色模式
- 设置会自动保存

## ⚠️ 常见问题

**Q: 转盘指针不准？**  
A: 已修复！现在转盘使用整圈旋转，指针100%对准选中的美食。

**Q: 图片加载很慢？**  
A: 所有41张图片都已经下载到本地 `images/` 文件夹，加载非常快！

**Q: 手机上摇一摇不工作？**  
A: 需要使用 HTTPS 或 localhost 访问，部分浏览器可能不支持陀螺仪 API。

**Q: 收藏和历史会丢失吗？**  
A: 数据保存在浏览器的 LocalStorage 中，除非清除浏览器数据，否则会一直保留。

**Q: 可以离线使用吗？**  
A: 完全可以！所有资源都在本地，无需网络连接。

**Q: 可以添加自己喜欢的美食吗？**  
A: 可以！编辑 `data.js` 文件，按照现有格式添加新的美食数据，并将图片放入 `images/` 文件夹。

## 🎨 自定义

### 添加新美食

编辑 `data.js`，在 `foodData` 数组中添加：

```javascript
{
    name: "你的美食名称",
    category: "分类",
    image: "images/your-image.jpg",
    description: "简短描述"
}
```

### 修改颜色主题

编辑 `styles.css`，修改 CSS 变量即可改变整体配色。

### 调整转盘大小

修改 `app.js` 中的 `canvas.width` 和 `canvas.height` 值。
