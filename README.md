# 今天吃什么 - 美食决策助手

一个帮你解决"选择困难症"的Web应用，通过转盘抽取和列表浏览来决定今天吃什么！

## ✨ 功能特点

### 🎰 转盘抽取
- 随机转盘抽取，增加趣味性
- 支持按分类筛选美食
- 精确的指针对齐，确保抽中结果准确
- 流畅的旋转动画

### 📋 美食列表
- 65+ 种中国常见美食
- 15 个美食分类（中式快餐、面食、小吃、川湘菜、火锅、烧烤、西餐、日韩料理等）
- 支持搜索功能
- 分类筛选
- 美食卡片展示，包含真实图片

### ⭐ 收藏功能
- 收藏喜欢的美食
- 持久化保存到浏览器

### 📜 历史记录
- 记录每次抽取结果
- 显示抽取时间
- 支持清空历史

### 🌙 深色模式
- 一键切换深色/浅色主题
- 自动保存偏好设置

### 📱 手机端优化
- 完全响应式设计
- 支持摇一摇抽取（需要手机浏览器支持）
- 触摸友好的操作界面

## 🚀 使用方法

1. 直接用浏览器打开 `index.html` 即可使用
2. 在转盘页面点击"开始抽取"按钮，转盘会自动旋转并选出一道美食
3. 可以通过筛选器选择特定分类的美食
4. 在列表页面可以浏览所有美食
5. 手机上可以摇一摇触发抽取

## 🎨 美食分类

- **中式快餐**: 黄焖鸡、盖浇饭、扬州炒饭、煲仔饭等
- **面食**: 兰州拉面、重庆小面、刀削面、热干面等
- **小吃**: 麻辣烫、酸辣粉、煎饼果子、肉夹馍等
- **川湘菜**: 烤鱼、水煮鱼、宫保鸡丁、回锅肉等
- **火锅**: 重庆火锅、海底捞、潮汕牛肉火锅等
- **烧烤**: 羊肉串、烤鸡翅、烤茄子等
- **西餐**: 牛排、披萨、意大利面、汉堡等
- **日韩料理**: 寿司、拉面、烤肉、石锅拌饭等
- **粤菜**: 早茶、烧鹅、肠粉等
- **甜品饮品**: 奶茶、咖啡、蛋糕、冰淇淋等
- **轻食**: 沙拉、三明治、健身餐等

## 🔧 技术栈

- **前端**: HTML5 + CSS3 + 原生 JavaScript
- **Canvas**: 用于绘制转盘
- **LocalStorage**: 用于保存收藏和历史
- **响应式设计**: 支持各种屏幕尺寸
